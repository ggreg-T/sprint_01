Bonjour !
Pour utiliser toute la puissance de ce site, veuillez suivre les instructions suivantes :
1- Les maquettes du site sont dans le dossier « makote »
2- Le fichiers du pseudo code est disponible dans la racine, le nom du fichier :pseudo_code.txt
3- Le site est dans le dossier « generateur »
4- Pour utiliser le site «lancer » (pas trop fort, sinon on décline toute responsabilité du mauvais fonctionnement qui en découlerait)
le fichier ftg1.html avec votre navigateur préféré (perso c’est Firefox).
5- Utilisation du site :
	Une liste de nom est déjà inscrite : vous pouvez « effacer » un ou plusieurs noms en cliquant sur les croix en face du nom choisi,
     et vous pouvez en ajouter d’autre, en utilisant la case prévu à cet effet.
	Puis vous choisissez le nombre de groupe que vous voulez.
	Et enfin cliquez sur « Générer »

Merci d’avoir lu ce mode d’emploi.
